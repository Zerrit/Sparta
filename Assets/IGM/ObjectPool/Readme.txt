Создаем пустой объект в сцене, на него вешаем ObjectPoolManager. 

В нем есть список ObjectPools, добавляем туда нужные объекты для пула. 
Name - название объекта. Названия не должны повторяться. 
Prefab - Сам объект который будет спавниться.
Count - сколько преспавленных объектов будет в сцене. 

Поле ObjectPoolEnumPath содержит путь до скрипта в который будут записываться все объекты из пула для их дальнейшего удобного выбора в инспекторе. По дефолту он стоит правельный, если не менять расположение папок, то его трогать не нужно.

После обновления пула объектов нажать на кнопку "Init". После чего автоматически произойдет перезагрузка скриптов и все будет готово к использованию.
